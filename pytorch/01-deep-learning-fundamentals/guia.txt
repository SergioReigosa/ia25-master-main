1.3 Simplificar las clases objetivo

Si tienes múltiples categorías en tu columna de etiquetas (por ejemplo, "diagnosis"), simplifícalas a 3 categorías: 'hyperthyroid', 'hypothyroid', y 'negative'.

def simplify_thyroid_class(diagnosis_string: str) -> str:
    if diagnosis_string in ['hyperthyroid', 'compensated hyperthyroid', 'subclinical hyperthyroid']:
        return 'hyperthyroid'
    elif diagnosis_string in ['hypothyroid', 'subclinical hypothyroid']:
        return 'hypothyroid'
    else:
        return 'negative'

df['diagnosis'] = df['diagnosis'].apply(simplify_thyroid_class)
1.4 Verificar la distribución de las clases

Asegúrate de que las clases estén distribuidas adecuadamente. Si hay desbalance, puedes aplicar técnicas como sobremuestreo o ponderación de clases.

print(df['diagnosis'].value_counts())
2️⃣ Preparar los datos para PyTorch
2.1 Convertir los datos a tensores de PyTorch

Convierte las características X y las etiquetas y en tensores de PyTorch.

Características (X): Las columnas de entrada.

Etiquetas (y): La columna de clase.

import torch
from sklearn.preprocessing import LabelEncoder

X = df.drop('diagnosis', axis=1).values  # Elimina la columna 'diagnosis'
y = df['diagnosis'].values  # Obtén las etiquetas

# Si y es texto, usa LabelEncoder
le = LabelEncoder()
y = le.fit_transform(y)  # Convierte las clases en números

# Convierte a tensores de PyTorch
X_tensor = torch.tensor(X, dtype=torch.float32)
y_tensor = torch.tensor(y, dtype=torch.long)
2.2 Dividir en conjunto de entrenamiento y prueba

Divide los datos en train/test para evaluar el modelo después de entrenarlo.

from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(X_tensor, y_tensor, test_size=0.2, random_state=42)
3️⃣ Definir y entrenar el modelo
3.1 Definir la arquitectura del modelo

Puedes crear un modelo sencillo de red neuronal en PyTorch usando nn.Sequential.

import torch.nn as nn

class ThyroidModel(nn.Module):
    def __init__(self, input_dim, hidden_dim, output_dim):
        super(ThyroidModel, self).__init__()
        self.fc1 = nn.Linear(input_dim, hidden_dim)
        self.relu1 = nn.ReLU()
        self.fc2 = nn.Linear(hidden_dim, output_dim)

    def forward(self, x):
        x = self.fc1(x)
        x = self.relu1(x)
        x = self.fc2(x)
        return x

# Parámetros
input_dim = X_train.shape[1]  # Número de características de entrada
hidden_dim = 64  # Número de neuronas en la capa oculta
output_dim = 3  # Número de clases

# Crear el modelo
model = ThyroidModel(input_dim, hidden_dim, output_dim)
3.2 Definir la función de pérdida y el optimizador

La función de pérdida debe ser CrossEntropyLoss para clasificación, y el optimizador puede ser Adam.

criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=0.001)
3.3 Entrenar el modelo

Entrena el modelo durante varias épocas. Guarda la pérdida de entrenamiento y validación.

train_loss = []
val_loss = []
num_epochs = 100

for epoch in range(num_epochs):
    model.train()
    optimizer.zero_grad()
    outputs = model(X_train)
    loss = criterion(outputs, y_train)
    loss.backward()
    optimizer.step()

    train_loss.append(loss.item())

    # Evaluación en datos de prueba
    model.eval()
    with torch.no_grad():
        val_outputs = model(X_test)
        val_loss_val = criterion(val_outputs, y_test)
        val_loss.append(val_loss_val.item())

    if (epoch + 1) % 5 == 0:
        print(f'Epoch [{epoch+1}/{num_epochs}], Loss Train: {loss.item():.4f}, Loss Test: {val_loss_val.item():.4f}')
4️⃣ Evaluación del modelo
4.1 Evaluar el rendimiento del modelo

Usa la matriz de confusión para ver cuántas predicciones fueron correctas y cuántas incorrectas.

from sklearn.metrics import confusion_matrix
import seaborn as sns
import matplotlib.pyplot as plt

model.eval()
with torch.no_grad():
    y_pred = model(X_test)
    _, y_pred = torch.max(y_pred, 1)

y_test = y_test.numpy()
y_pred = y_pred.numpy()

cm = confusion_matrix(y_test, y_pred)

sns.heatmap(cm, annot=True, fmt="d", cmap="Blues", xticklabels=le.classes_, yticklabels=le.classes_)
plt.title("Matriz de Confusión")
plt.xlabel("Predicciones")
plt.ylabel("Reales")
plt.show()
4.2 Análisis de resultados

Revisa la diagonal principal de la matriz de confusión para ver las predicciones correctas, y las fuera de la diagonal para los errores cometidos.

5️⃣ Mejoras posibles

Ajustar hiperparámetros:

Experimenta con diferentes tamaños de capas ocultas o tasa de aprendizaje.

Rebalancear clases:
Si tienes desbalanceo en las clases (por ejemplo, más muestras de negative que de hypothyroid o hyperthyroid), puedes usar técnicas de rebalanceo (como undersampling, oversampling, o ajustar los pesos de clase).

Early Stopping:
Implementa una técnica de detención temprana para evitar el overfitting.

Regularización:
Usa Dropout o L2 regularization para prevenir el sobreajuste.
